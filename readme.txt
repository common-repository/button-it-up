Button It Up changes the upload/insert button style to look like an actual button.

== Description ==

Make clickable things look clickable. Button It Up changes the upload/insert button style to look like an actual button, making the button more findable for first-time WordPress users. 

Read more about the reasoning behind this simple UI change and the beneifits to the end user: http://www.parisvega.com/if-ya-wanna-click-it-ya-better-put-a-button-it/ 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder named `button-it-up` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the post/page editor screen to see the new button style of the Upload/Insert button.
